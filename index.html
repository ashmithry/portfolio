Projects

Hands-on applications of my skills in software development, AI, and game design.

DECA Forge

A comprehensive management platform to automate operations for my 530-member DECA chapter. It's built to handle event registration, track member participation, and streamline communication, solving our 198:1 student-to-advisor bottleneck.

Web Dev

The FACTory

An educational game on media literacy co-developed during my internship at the Virginia Serious Games Institute. The game teaches students to identify misinformation and is currently deployed in local elementary schools. Our abstract was published and presented at two university symposia.

Game Dev

EcoSpark

A "serious game" from Cognicade Studios that teaches financial literacy and environmental sustainability. Players must manage a budget while making eco-conscious choices, learning about their interconnectedness in a closed-feedback loop.

Game Dev

The Rookie

A first-person spy platformer inspired by Karlson and Portal. Players use grappling, hacking, and energy blasting mechanics, only to discover the entire game is a simulation test they must escape. This project was a deep dive into complex physics and narrative design.

Game Dev

Grading Hat

An intelligent grading system that uses AI to assist educators. It provides consistent evaluation standards, integrates into existing workflows, and offers automated feedback generation to save teachers time and reduce bias.

AI/ML

Christmas Chaos

A fast-paced, top-down arcade shooter where you play as Santa Claus. Players must protect the Golden Present from an incoming horde of enemies using a bow, arrow, and sword, testing their reflexes in a survival format.

Game Dev

Moneytree Simulator

A rudimentary financial simulation where you are placed in the shoes of a random person. You must make the best possible investment and money allocation choices to achieve a high score, all while preparing for random life events.

Simulation

Plundr.io

A concept for a web-based, multiplayer .io game with a pirate theme. The project involved planning real-time networking, authoritative server logic, and simple, scalable 2D graphics for browser-based play.

Game Dev

Bread to Dough

A mobile game concept focused on running a bakery. The core loop involved resource management, customer orders, and expanding your business. This project was an exercise in designing for a mobile-first user experience.
